## Особенности

### Основная функциональность

| Особенность | Описание |
|---------|-------------|
| **Быстрый анализ сохранения** | Одна из самых быстрых программ для чтения сохраненных файлов |
| **Управление игроками** | Просматривайте, редактируйте, переименовывайте, меняйте уровень, разблокируйте технологии и управляйте игроками |
| **Управление гильдией** | Создавайте, переименовывайте, перемещайте игроков, разблокируйте лабораторные исследования и управляйте гильдиями |
| **Pal Editor** | Полный редактор статистики, навыков, IVs, ранга, душ, пола, переключения босса/счастливчика |
| **Инструменты базового лагеря** | Экспорт, импорт, клонирование, настройка радиуса и управление базами |
| **Просмотр карт** | Интерактивная карта базы и игрока с координатами и деталями |
| **Перенос персонажа** | Перенос персонажей между разными мирами/серверами (кросс-сохранение) |
| **Сохранить конверсию** | Преобразование между форматами Steam и GamePass |
| **Настройки мира** | Редактировать настройки WorldOption и LevelMeta |
| **Инструменты для создания временных меток** | Исправление отрицательных временных меток и сброс времени игроков |

### Универсальные инструменты

Пакет **Инструменты «Все в одном»** обеспечивает комплексное управление сохранениями:

- **Инструменты удаления**
  - Удаление игроков, баз или гильдий
  - Удаление неактивных игроков на основе временных ограничений.
  - Удаление дубликатов игроков и пустых гильдий
  - Удаление неиспользуемых/потерянных данных.

- **Инструменты очистки**
  - Удаление недействительных/модифицированных элементов.
  - Удалить неверные pals и passives.
  - Исправлен недопустимый pals (ограничение разрешенных максимальных характеристик)
  - Удаление недействительных структур.
  - Сброс зенитных турелей
  - Разблокировать private chests

- **Инструменты гильдии**
  - Восстановить все гильдии
  - Перемещайте игроков между гильдиями.
  - Сделать игрока лидером гильдии
  - Переименование гильдий
  - Максимальный уровень гильдии
  - Разблокировать все лабораторные исследования

- **Инструменты игрока**
  - Редактировать статистику и навыки игрока pal.
  - Разблокировать все технологии
  - Разблокировать смотровую клетку
  - Повышение/понижение уровня игроков
  - Переименовывать игроков

- **Сохранить утилиты**
  - Сброс миссий
  - Сброс подземелий
  - Исправлены временные метки
  - Обрезать переполненные запасы.
  - Генерация команд PalDefender

## Установка

### Предварительные условия

**Для автономного режима (Windows):**
- Windows 10/11
- [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170#latest-microsoft-visual-c-redistributable-version) (2015-2022 гг.)

**Для запуска из исходного кода (все платформы):**
- Python 3.11 или выше

### Автономный (Windows — рекомендуется)

1. Загрузите последнюю версию с [GitHub Releases](https://github.com/deafdudecomputers/PalworldSaveTools/releases/latest).
2. Распакуйте zip-файл.
3. Запустите `PalworldSaveTools.exe`.

### Из исходного кода (все платформы)

Стартовые сценарии автоматически создают виртуальную среду и устанавливают все зависимости.

**Использование uv:**
```bash
git clone https://github.com/deafdudecomputers/PalworldSaveTools.git
cd PalworldSaveTools
uv venv --python 3.12
uv run start.py
```

**Окна:**
```bash
git clone https://github.com/deafdudecomputers/PalworldSaveTools.git
cd PalworldSaveTools
start_win.cmd
```

**Линукс:**
```bash
git clone https://github.com/deafdudecomputers/PalworldSaveTools.git
cd PalworldSaveTools
chmod +x start_linux.sh
./start_linux.sh
```

### Филиалы

- **Стабильная** (рекомендуется): `git clone https://github.com/deafdudecomputers/PalworldSaveTools.git`
- **Бета** (последние функции): `git clone -b beta https://github.com/deafdudecomputers/PalworldSaveTools.git`

## Быстрый старт

1. **Загрузите сохранение**
   - Нажмите кнопку меню в шапке.
   – Выберите **Загрузить сохранение**.
   - Перейдите в папку сохранения Palworld.
   - Выберите `Level.sav`

2. **Изучите свои данные**
   - Используйте вкладки для просмотра игроков, гильдий, баз или карты.
   - Поиск и фильтрация, чтобы найти конкретные записи

3. **Внесите изменения**
   - Выберите элементы для редактирования, удаления или изменения.
   - Щелкните правой кнопкой мыши, чтобы открыть контекстное меню с дополнительными опциями.

4. **Сохраните изменения**
   - Нажмите кнопку меню → **Сохранить изменения**.
   - Резервные копии создаются автоматически

## Обзор инструментов

### Универсальные инструменты (AIO)

Основной интерфейс для комплексного управления сохранениями с пятью вкладками:

**Вкладка «Инвентарь игрока»** — просмотр и управление инвентарем всех игроков на сервере.
- Редактировать статистику игрока, инвентарь, экипированное снаряжение.
- Возможность редактировать качество, добавлять, удалять что-либо из инвентаря и экипированного снаряжения.

**Вкладка «Инвентарь базы»** — просмотр и управление инвентарем всех баз на сервере.
- Редактировать базовый инвентарь.
- Возможность очищать контейнеры, редактировать качество, добавлять, удалять что-либо из базового инвентаря.
- Включена фильтрация на предмет возможного мошенничества.

**Вкладка «Игроки»** — просмотр и управление всеми игроками на сервере.
- Редактируйте имена игроков, уровни и количество pal.
- Удаление неактивных игроков
- Просмотр гильдий игроков и последнего времени онлайн

**Вкладка «Гильдии»** — управляйте гильдиями и их базами.
- Переименовывать гильдии, менять лидеров
- Просмотр базовых локаций и уровней
- Удаление пустых или неактивных гильдий

**Вкладка «Базы»** – просмотр всех базовых лагерей.
- Экспорт/импорт базовых чертежей
- Клонировать базы в другие гильдии
- Отрегулируйте базовый радиус

### Просмотр карт

Интерактивная визуализация вашего мира:
- Просмотр всех базовых локаций и позиций игроков
- Фильтровать по гильдии или имени игрока
- Нажмите маркеры для получения подробной информации.
- Генерация команд `killnearestbase` для PalDefender.

### Перенос персонажей

Перенос персонажей между разными мирами/серверами (кросс-сохранение):
- Трансфер одного или всех игроков
- Сохраняет персонажей, pals, инвентарь и технологии.
- Полезно для перехода между кооперативными и выделенными серверами.

### Исправить сохранение хоста

Поменяйте UID между двумя игроками:
- Передача прогресса от одного игрока к другому
- Необходим для переноса хоста/кооператива на сервер.
- Полезно для смены роли хоста между игроками.
- Полезно для смены платформ (Xbox ↔ Steam)
- Решает проблемы с назначением UID хоста/сервера.
- **Примечание:** Затронутый игрок должен сначала создать персонажа в целевом сохранении.

## Руководства

### Сохранить расположение файлов

**Хозяин/Кооператив:**
```
%localappdata%\Pal\Saved\SaveGames\YOURID\RANDOMID\
```

**Выделенный сервер:**
```
steamapps\common\Palworld\Pal\Saved\SaveGames\0\RANDOMSERVERID\
```

### Разблокировка карты

<details>
<summary>Нажмите, чтобы развернуть инструкции по разблокировке карты</summary>

1. Скопируйте `LocalData.sav` из `resources\`.
2. Найдите папку сохранения на своем сервере/мире.
3. Замените существующий `LocalData.sav` скопированным файлом.
4. Запустите игру с полностью разблокированной картой.

> **Примечание.** Используйте инструмент **Восстановить карту** на вкладке «Инструменты», чтобы применить разблокированную карту ко ВСЕМ вашим мирам/серверам одновременно с автоматическим резервным копированием.

</details>

### Хост → Перенос сервера

<details>
<summary>Нажмите, чтобы развернуть руководство по переносу с хоста на сервер</summary>

1. Скопируйте папки `Level.sav` и `Players` из сохранения хоста.
2. Вставьте в папку сохранения выделенного сервера.
3. Запустить сервер, создать нового персонажа.
4. Подождите автосохранения, затем закройте
5. Используйте **Fix Host Save** для переноса GUID.
6. Скопируйте файлы обратно и запустите.

**Использование сохранения Fix Host:**
- Выберите `Level.sav` из временной папки.
- Выберите **старого персонажа** (из исходного сохранения).
- Выберите **нового персонажа** (вы только что создали)
– Нажмите **Перенести**.

</details>

### Смена хоста (смена хоста)

<details>
<summary>Нажмите, чтобы развернуть руководство по замене хоста</summary>

**Справочная информация:**
- Хост всегда использует `0001.sav` — тот же UID для любого хостера.
- Каждый клиент использует уникальный обычный UID (например, `123xxx.sav`, `987xxx.sav`).

**Предварительные требования:**
Оба игрока (старый и новый хост) должны иметь свои обычные сохранения. Это происходит путем присоединения к миру хозяина и создания нового персонажа.

**Шаги:**

1. **Убедитесь, что существуют регулярные сохранения**
   - Игрок А (старый хост) должен иметь обычное сохранение (например, `123xxx.sav`).
   - Игрок Б (новый хост) должен иметь обычное сохранение (например, `987xxx.sav`).

2. **Замените сохранение старого хоста на обычное**
   - Используйте PalworldSaveTools **Исправить сохранение хоста**, чтобы поменять местами:
   - `0001.sav` старого хоста → `123xxx.sav`
   - (Это перемещает прогресс старого хоста из слота хоста в слот обычного игрока)

3. **Замените обычное сохранение нового хоста на сохранение хоста**
   - Используйте PalworldSaveTools **Исправить сохранение хоста**, чтобы поменять местами:
   - Новый хост `987xxx.sav` → `0001.sav`.
   - (Это перемещает прогресс нового хоста в слот хоста)

**Результат:**
- Игрок Б теперь является хозяином со своим персонажем и pals в `0001.sav`.
- Игрок А становится клиентом со своим первоначальным прогрессом в `123xxx.sav`.

</details>

### Базовый экспорт/импорт

<details>
<summary>Нажмите, чтобы развернуть базовое руководство по экспорту/импорту</summary>

**Экспорт базы:**
1. Загрузите сохранение в формате PST.
2. Перейдите на вкладку «Базы».
3. Щелкните базу правой кнопкой мыши → Экспортировать базу.
4. Сохранить как файл `.json`.

**Импорт базы:**
1. Перейдите на вкладку «Базы» или в «Просмотр карт базы».
2. Щелкните правой кнопкой мыши по гильдии, в которую вы хотите импортировать базу.
3. Выберите базу импорта.
4. Выберите экспортированный файл `.json`.

**Клонирование базы:**
1. Щелкните базу правой кнопкой мыши → Клонировать базу.
2. Выберите целевую гильдию
3. База будет клонирована со смещением позиционирования.

**Регулировка базового радиуса:**
1. Щелкните базу правой кнопкой мыши → Настроить радиус.
2. Введите новый радиус (50–1000 %).
3. Сохраните и загрузите сохранение в игре для переназначаемых структур.

</details>

## Устранение неполадок

### "VCRUNTIME140.dll не найден"

**Решение:** Установите [Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170#latest-microsoft-visual-c-redistributable-version).

### `struct.error` при разборе сохранить

**Причина:** Устаревший формат файла сохранения.

**Решение:**
1. Загрузите сохранение в игру (режим «Одиночка», «Кооператив» или «Выделенный сервер»).
2. Это запускает автоматическое обновление структуры.
3. Убедитесь, что сохранение было обновлено после выхода последнего обновления игры.

### GamePass конвертер не работает

**Решение:**
1. Закройте версию GamePass Palworld.
2. Подождите несколько минут.
3. Запускаем преобразователь Steam → GamePass.
4. Запустите Palworld на GamePass для проверки.
